## Node Type: Array

Represents a concrete array as the parse result of a [type_array](../specs/type_array.md)

Used for clarity in examples of [parse_type](../specs/parse_type.md)

### S-expression format
```
(array 0 1 2)
```

Each item after "array" is the data parsed according to the [type_array](../specs/type_array.md)
